Icon Files Needed
=================

Please add the following icon files to this directory:

1. icon16.png  - 16x16 pixels
2. icon48.png  - 48x48 pixels
3. icon128.png - 128x128 pixels

These icons will be displayed:
- icon16: Extension toolbar icon
- icon48: Extension management page
- icon128: Chrome Web Store (if publishing)

Recommended Design:
- Simple, recognizable symbol related to hotels/bookings
- Clear at small sizes
- Matches NewBook/hotel branding if desired

Temporary Workaround:
Until you add real icons, you can:
1. Copy the icons from the old extension (chrome-hotel-link-extention/icons/)
2. Or create simple placeholder icons online at: https://www.favicon-generator.org/

The extension will still work without icons, but Chrome will show a default placeholder.
